History Feature Disabled

Supabase Configuration Required

The transcription history feature requires Supabase to store your transcription records.

To enable this feature:
  1. Create a free Supabase account at supabase.com
  2. Create a new project
  3. Create a table named transcriptions with the following columns:
    • id (type: uuid, primary key)
    • file_name (type: text)
    • language (type: text)
    • model (type: text)
    • task (type: text)
    • format (type: text)
    • preview (type: text)
    • created_at (type: timestamp with time zone, default: now())
  4. Get your Supabase URL and API key from the project settings
  5. Set the environment variables SUPABASE_URL and SUPABASE_KEY
